# We's Volleyball Manager

チームを一つにまとめ、練習を続けたくなる仕組みを作る。

## Overview

We's Volleyball Manager は、Flutter と Firebase を用いて開発したバレーボールチーム運営アプリです。

代表としてチーム運営を行う中で、

* 練習や試合の出欠確認
* 予定の共有
* 選手情報の管理

を LINE 上で行っていましたが、

* 予定の把握漏れ
* 参加人数が分からない
* 情報が分散してしまう

といった課題がありました。

そこで、

* チーム運営の効率化
* 情報の一元管理
* 選手の成長の可視化
* モチベーション向上

を目的として、本アプリの開発を行いました。

## Features

### Player Management

* 選手情報管理
* 身体データ管理
* 能力値管理
* ポジション適性算出

### Schedule Management

* カレンダー表示
* 練習予定登録
* 試合予定登録
* 出欠登録
* 遅刻時間登録
* 遅刻時間未定機能
